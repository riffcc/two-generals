# Two Generals Protocol

**Status:** Coming Soon

A practical solution to the Two Generals Problem using cryptographic proof stapling.

---

## What This Is

A protocol that achieves **guaranteed symmetric coordination** over unreliable networks.

Both parties either:
- ✅ Both proceed (coordinated attack)
- ✅ Both abort (coordinated retreat)

Never:
- ❌ One proceeds, one aborts (disaster)
